Description

A split-view interface for reviewing, editing, and approving AI-extracted data against an original Statement of Work document. Supports human-in-the-loop verification workflows.

Design System Alignment

Implements the split-screen pattern using grid tokens, uses smart inputs for AI confidence visualization, and adheres to the primary teal color palette.

Responsive Specifications

Mobile Adaptations

  • Layout switches from 2-column grid to single-column stack
  • Document viewer gets collapsible behavior or fixed height
  • Action bar becomes sticky at bottom

Tablet Adaptations

  • Maintains split view but with reduced margins
  • Typography scales down slightly

Desktop Enhancements

  • Independent scrolling for document and form panels
  • Fixed height container to fit viewport
  • Hover states on confidence indicators
